# Sauls Object Registry Bundle

Object registry Symfony bundle. 

## Requirements

* php >= 7.2
* Symfony >= 4.1

## Installation

### Using composer

```bash
$ composer require sauls/object-registry-bundle
```

### Add manually to your composer.json file

```json
{
  "require": {
    "sauls/object-registry-bundle": "^1.0" 
  }
}
```
